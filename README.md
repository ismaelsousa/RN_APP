<h2 align='center'> # This project is to learn react-native framework </h2>

[React-Native](https://reactnative.dev)
  - Version ~> "0.64.1"

## 🏗 Running

- Pre running (Android|Ios)
  - run `npm install`
- IOS:
  - run `npx pod-install && npx react-native run-ios`
- Android:   
  - run `npx react-native run-android`

## Folder structure
